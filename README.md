# 2D Unity Game Project (Guided Coursework)

This repository contains the C# scripts for a 2D game developed during my programming course. Since the Unity editor environment has been archived, the source code is hosted here to demonstrate my understanding of game logic and object-oriented programming in C#.

### 🛠️ Key Scripts & Logic Included:
* **Player Movement:** Handles input detection and physics-based movement.
* **Collision Detection:** Manages how game objects interact (e.g., triggers when hitting enemies or collecting items).
* **Game Manager:** Tracks the game states such as scoring and player health.
